# KindMeal-Website-Clone
Project: make a clone of https://www.kindmeal.my/ site

*This is an online food website which speciaizes in selling unique food all over the globe. The website provides meat free food for only vegetarian peoples.*


## Technology Stack Used:
- HTML5
- CSS
- JavaScript
